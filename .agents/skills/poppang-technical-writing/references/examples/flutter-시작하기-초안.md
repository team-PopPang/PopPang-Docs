# Flutter 사용법

Flutter를 설치하고 프로젝트를 만들면 앱을 실행할 수 있습니다. 이 문서에서는 Flutter를 대충 설명하고 설치도 하고 프로젝트도 만들고 실행도 해보겠습니다. 중간에 안 되면 알아서 검색하면 됩니다.

## 준비

macOS나 Windows에서 설치할 수 있습니다. 버전은 최신이면 되고 Android Studio나 VS Code도 설치하면 좋습니다.

## Flutter 설치

Flutter SDK를 받고 PATH를 잡습니다. 자세한 것은 공식 문서를 참고하면 되는데 여기서는 생략합니다.

## 프로젝트 만들기

터미널에서 프로젝트를 만들고 실행합니다.

```bash
flutter create my_app
cd my_app
flutter run
```

## 설명

Hot reload는 코드 변경을 빨리 반영해 줍니다. 상태 관리나 위젯 구조 같은 것은 나중에 공부해도 됩니다. 처음에는 그냥 실행만 되면 됩니다. 여러 환경에서 설정이 다를 수 있으니 잘 안 되면 각자 환경을 확인해 보세요.
